01 / FILOSOFIÝA

# Spacing — görünmeýän skelet

Reňk we tipografiýa gözle görünýär, ýöne spacing — duýulýar. Erbet spacing bilen sahypa "duýgur däl" görünýär, gowy spacing bilen "premium" duýulýar.

## Möhüm konsept — Spacing vs Sizing

Iki dürli ölçeg kategoriýasy bar:

- SPACING tokenleri (space-1, space-2...) — padding, gap, margin üçin
- SIZING tokenleri (size-32, size-48, size-56...) — komponent height, width üçin
Bularyň gymmatlyklary meňzeş bolup biler (ikisi-de 4px-iň göterimi), ýöne niýetlerini bulaşdyrmaýyn. Bu mesele köp dizaýn ulgamlarynda bar.

04 / SIZING SCALE — TÄZE

# Sizing — komponent ölçegleri

Component HEIGHT we WIDTH üçin aýratyn scale. Bu — padding/gap/margin DÄL. Sizing tokenleri spacing-den uly bolup biler we başga gymmatlyklary öz içine alyp biler (mysal: 56, 80, 240) — ýöne 4px grid-de bolmaly.



## Sizing — spacing-den näme tapawudy?

- Sizing tokenleri komponent height/width üçin niýetlenen — element ölçegi
- Hem-de spacing scale-de bolmadyk gymmatlyklar sizing-de bolup biler (36, 44, 56, 80, 120)
DÜZGÜN — Sizing tokeni-de 4px grid bolmaly. 33px, 47px, 65px DÄL — 32, 48, 64 saýla.

DÜZGÜN — Spacing tokeni padding/gap/margin DÄL ulanma. Sizing tokenini element içine padding hökmünde DÄL ulan.


05 / DÜZGÜNLER

# Padding · Gap · Margin

Köp adamyň bulaşdyrýan ýeri. Pro hökmünde anyk düzgün.



## DOĞRY ulanyş mysallary

- Button: padding space-3 (Y) + space-4 (X) → içki rahatlyk
- List: gap space-3 between items
- Section: margin-top space-6
- Screen: padding-X space-4 (DEFAULT 16px)

## NÄDOĞRY ulanyş

- List item-de margin → list üçin gap ulan
- Section padding + screen margin garyşygy → bulaşyk
- Random sany 17px, 23px → scale-de ýok
DÜZGÜN — Element içi → padding. Çagalary arasy → gap. Daşky → margin. KARYŞDYRMA.
